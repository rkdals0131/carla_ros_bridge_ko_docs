Carla Twist to Control

carla_twist_to_control 패키지는 geometry_msgs.Twist를 carla_msgs.CarlaEgoVehicleControl로 변환합니다.

ROS API

구독 (Subscriptions)

토픽 타입 설명
/carla/<ROLE NAME>/vehicle_info carla_msgs.CarlaEgoVehicleInfo 에고 차량 정보, 최대 조향각 수신용.
/carla/<ROLE NAME>/twist geometry_msgs.Twist 변환할 Twist 메시지.


발행 (Publications)

토픽 타입 설명
/carla/<ROLE NAME>/vehicle_control_cmd carla_msgs.CarlaEgoVehicleControl 변환된 차량 제어 명령.
